## Week 4 Achievements

- Understood AWS security mechanisms and learned how to apply:
  - IAM Permission Boundary for controlling user permissions.
  - AWS KMS for encryption and key management.
  - AWS Security Hub for security monitoring and compliance checks.
  - AWS WAF for protecting web applications from common attacks.

- Built a resource monitoring system using Amazon CloudWatch integrated with Grafana.
  - Learned how to collect and visualize system metrics.
  - Monitored the performance and operational status of AWS services.

- Became familiar with AWS Lambda and AWS Systems Manager.
  - Learned how to automate administrative tasks.
  - Improved understanding of serverless operations and infrastructure management.

- Successfully deployed Docker applications on Amazon ECS.
  - Understood the process of running containerized applications on AWS.
  - Learned the basic workflow of automated deployment using AWS CodePipeline.

- Learned AWS cost optimization strategies, including:
  - Auto Scaling for dynamically adjusting resources.
  - Savings Plans and Reserved Instances for reducing long-term costs.
  - AWS cost monitoring and analysis.

- Gained practical experience in managing AWS infrastructure with a focus on security, monitoring, automation, deployment, and cost efficiency.
