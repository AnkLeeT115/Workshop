## Kết quả đạt được tuần 9

- Hiểu tổng quan hệ thống **Genuine Scale Models E-commerce Backend**:
  - Hiểu kiến trúc Serverless.
  - Nắm vai trò của frontend, backend và các dịch vụ AWS.
  - Hiểu quy trình quản lý sản phẩm, tồn kho và xử lý đơn hàng.

- Thiết lập thành công môi trường phát triển:
  - Cấu hình AWS CLI và AWS SAM.
  - Cài đặt các công cụ cần thiết.
  - Kiểm tra backend trên môi trường local.

- Hiểu vai trò của các dịch vụ AWS trong hệ thống:
  - **Amazon S3**: Lưu trữ hình ảnh sản phẩm và tài nguyên tĩnh.
  - **Amazon CloudFront**: Phân phối nội dung nhanh chóng.
  - **Amazon API Gateway**: Quản lý API.
  - **AWS Lambda**: Xử lý logic backend.
  - **Amazon DynamoDB**: Lưu trữ sản phẩm, đơn hàng, người dùng và tồn kho.
  - **Amazon Cognito**: Xác thực và phân quyền người dùng.

- Hiểu các quy trình trong hệ thống thương mại điện tử:
  - Quản lý catalog sản phẩm.
  - Kiểm tra tồn kho.
  - Xử lý đơn hàng.
  - Quản lý tài khoản người dùng.

- Thiết kế cấu trúc dữ liệu cơ bản:
  - Thông tin sản phẩm.
  - Đơn hàng khách hàng.
  - Tài khoản người dùng.
  - Quản lý tồn kho.

- Làm quen với quy trình phát triển phần mềm:
  - Xác thực bằng Amazon Cognito.
  - Git workflow.
  - Coding convention.
  - Quy trình triển khai Serverless.
