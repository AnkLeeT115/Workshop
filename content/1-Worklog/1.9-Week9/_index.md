## Week 9 Achievements

- Gained an overview of the **Genuine Scale Models E-commerce Backend** system:
  - Understood Serverless architecture.
  - Learned the responsibilities of frontend, backend services, and AWS components.
  - Understood the workflow of product management, inventory, and order processing.

- Successfully prepared the development environment:
  - Configured AWS CLI and AWS SAM.
  - Installed required development tools.
  - Tested backend services locally.

- Understood the roles of AWS services used in the system:
  - **Amazon S3**: Stores product images and static assets.
  - **Amazon CloudFront**: Provides fast content delivery.
  - **Amazon API Gateway**: Handles API requests.
  - **AWS Lambda**: Executes backend business logic.
  - **Amazon DynamoDB**: Stores products, orders, users, and inventory data.
  - **Amazon Cognito**: Provides user authentication and authorization.

- Learned e-commerce system workflows:
  - Product catalog management.
  - Inventory checking.
  - Order processing.
  - User account management.

- Designed basic database structures:
  - Product information.
  - Customer orders.
  - User accounts.
  - Inventory management.

- Became familiar with software development practices:
  - Authentication using Amazon Cognito.
  - Git workflow.
  - Coding conventions.
  - Serverless deployment process.
