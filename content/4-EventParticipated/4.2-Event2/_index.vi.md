# Báo cáo thu hoạch: "Technology and Career Workshop"

### Mục tiêu của sự kiện

- Tìm hiểu cách kết nối Godot Client với AWS WebSocket.
- Hiểu về công nghệ container hóa và Docker.
- Khám phá ứng dụng Machine Learning trong Hệ thống phát hiện xâm nhập mạng (NIDS).
- Tìm hiểu lộ trình nghề nghiệp của System Administrator, Cloud Computing và DevOps.
- Hiểu mô hình GraphRAG với Amazon Bedrock và Amazon Neptune.
- Nâng cao kỹ năng làm việc nhóm và giao tiếp trong môi trường công nghệ.

### Diễn giả

- **Nguyễn Quốc Bảo** – Connecting Godot Clients with AWS WebSockets
- **Bảo Huỳnh** – Docker: A Containerization Technology
- **Lê Hoàng Gia Đại** – Machine Learning-Based NIDS on AWS
- **Trần Trung Vinh** – Từ IT Helpdesk đến Senior Sysadmin, Cloud và DevOps
- **Việt Phát** – GraphRAG với Amazon Bedrock và Amazon Neptune
- **Trương Huy Phước** – The Art of Effective Teamwork

### Nội dung nổi bật

#### Kết nối Godot Client với AWS WebSocket

- Giới thiệu các khái niệm về Multiplayer Networking.
- So sánh HTTP Polling, WebSocket và UDP/ENet.
- Trình bày kiến trúc game Multiplayer trên nền tảng AWS.
- Giải thích vai trò của API Gateway, DynamoDB và AWS Lambda.
- Demo kết nối Godot với AWS WebSocket.
- Trình diễn hệ thống Matchmaking.
- Đề xuất sử dụng AWS GameLift để mở rộng hệ thống game.

#### Docker – Công nghệ Container hóa

- Giới thiệu Virtualization và Containerization.
- So sánh máy ảo và Container.
- Giải thích kiến trúc Docker và Dockerfile.
- Giới thiệu Image, Container và Image Layer.
- Demo các lệnh Docker phổ biến.
- Chia sẻ các tình huống ứng dụng Docker trong thực tế.

#### Hệ thống phát hiện xâm nhập mạng bằng Machine Learning trên AWS

- Giới thiệu AWS WAF và NIDS sử dụng Machine Learning.
- Phân tích hạn chế của hệ thống bảo mật dựa trên Rule.
- Sử dụng bộ dữ liệu CSE-CIC-IDS2018 để huấn luyện mô hình.
- Trình bày quy trình tiền xử lý dữ liệu và đánh giá mô hình.
- Giới thiệu kiến trúc giám sát bảo mật trên AWS.
- Demo Dashboard phát hiện tấn công mạng.

#### Từ IT Helpdesk đến Senior Sysadmin, Cloud và DevOps

- Chia sẻ hành trình phát triển nghề nghiệp từ IT Helpdesk.
- Giới thiệu kỹ năng và trách nhiệm của System Administrator.
- Trình bày quá trình chuyển đổi từ hệ thống On-premises lên Cloud.
- Giải thích mối liên hệ giữa System Administration và DevOps.
- Chia sẻ kinh nghiệm phỏng vấn và phát triển nghề nghiệp.
- Khuyến khích học tập thông qua các dự án thực tế.

#### Xây dựng GraphRAG với Amazon Bedrock và Amazon Neptune

- Giới thiệu Retrieval-Augmented Generation (RAG).
- Phân tích hạn chế của RAG truyền thống.
- Giải thích GraphRAG trong việc xử lý các truy vấn nhiều bước.
- So sánh kiến trúc GraphRAG Managed và Custom.
- Giới thiệu Amazon Bedrock và Amazon Neptune.
- Demo quy trình xây dựng GraphRAG trên AWS.

#### Nghệ thuật làm việc nhóm hiệu quả

- Giới thiệu nền tảng của làm việc nhóm.
- Trình bày Four Golden Rules.
- So sánh hiệu quả làm việc cá nhân và làm việc nhóm.
- Giới thiệu các công cụ hỗ trợ cộng tác trực tuyến.
- Chia sẻ phương pháp nâng cao hiệu quả làm việc nhóm.

### Những kiến thức tiếp thu được

#### Multiplayer Networking

- WebSocket hỗ trợ giao tiếp thời gian thực cho các ứng dụng Multiplayer.
- API Gateway, DynamoDB và Lambda có thể xây dựng hệ thống Matchmaking.
- AWS GameLift hỗ trợ triển khai và mở rộng Game Server.

#### Docker và Container

- Container nhẹ hơn máy ảo và dễ triển khai.
- Dockerfile giúp tái tạo môi trường ứng dụng một cách nhất quán.
- Docker được sử dụng rộng rãi trong phát triển và triển khai phần mềm.

#### Machine Learning và An toàn thông tin

- Hệ thống dựa trên Rule không thể phát hiện mọi cuộc tấn công.
- Machine Learning giúp nâng cao khả năng phát hiện xâm nhập.
- Dữ liệu chất lượng và đánh giá mô hình là yếu tố quan trọng.

#### Định hướng nghề nghiệp

- System Administrator cần kỹ năng kỹ thuật và khả năng xử lý sự cố.
- Cloud Computing và DevOps là những hướng phát triển tiềm năng.
- Kinh nghiệm thực tế từ các dự án là lợi thế khi tìm việc.

#### GraphRAG

- RAG truyền thống gặp khó khăn với các truy vấn nhiều bước.
- GraphRAG khai thác mối quan hệ giữa các dữ liệu để tăng độ chính xác.
- Amazon Bedrock và Neptune hỗ trợ hiệu quả việc xây dựng GraphRAG.

#### Làm việc nhóm

- Giao tiếp rõ ràng giúp nâng cao hiệu quả cộng tác.
- Các thành viên cần thống nhất mục tiêu và trách nhiệm.
- Công cụ cộng tác giúp tăng năng suất làm việc.

### Ứng dụng vào học tập và công việc

- Xây dựng thử nghiệm ứng dụng Multiplayer bằng Godot và AWS WebSocket.
- Thực hành Docker và xây dựng Dockerfile cho các dự án cá nhân.
- Tìm hiểu Machine Learning trong lĩnh vực An toàn thông tin.
- Cải thiện kiến thức về Linux, Networking và Cloud Computing.
- Xây dựng một ứng dụng GraphRAG đơn giản trên AWS.
- Áp dụng kỹ năng làm việc nhóm vào các dự án học tập và phát triển phần mềm.
